1.Công cụ AI:chatgpt
2.Các prompt chính đã sử dụng:
-Yêu cầu chatgpt tạo ảnh
-Gợi ý các làm web sách cơ bản
3.Những phần tự thực hiện:
-Sau khi đã tìm hiểu một trang web sách cần có gì,

ngoại trừ JS là ngôn ngữ mới.
4.Những phần tham khảo AI:
-JS thực hiện 2 chức năng hiển thông báo tin nhắn và 
in ra chi tiết sách đã chọn lỗi  nhiều cần lên AI sửa lỗi.
-Và đổi màu khi chọn menu cách làm tương tự C :tham khảo code sửa lỗi cho đúng.
5.Những khó khăn gặp phải:
-Khi học C chuyển sang JS thường viết sai cú pháp.
-Thiết kế web sách đẹp làm mất khá nhiều thời gian.
